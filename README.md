# Blog Post App

A full-stack web application built with Laravel and Bootstrap that allows users to create, manage, and search blog posts with authentication and image upload support.

## 🔍 Overview

This project is a CRUD-based blog platform where users can register, log in, and manage their own posts. It demonstrates core Laravel concepts including authentication, database relationships, and file handling, combined with a responsive UI.

## ✨ Features

* **User Authentication** – Secure registration, login, and logout
* **CRUD Operations** – Create, edit, view, and manage blog posts
* **Image Upload** – Upload and validate images (JPG/PNG)
* **Search Functionality** – Search posts by title
* **Pagination** – Organized post listing

## 🛠️ Tech Stack

**Backend**

* Laravel 12
* PHP 8.2
* SQLite

**Frontend**

* Blade Templates
* Bootstrap

## 🚀 Run Locally

```
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
npm run dev
php artisan serve
```
